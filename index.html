About this project

NutriTrack is a data-driven REST API built with FastAPI and SQLite, backed by the USDA FoodData Central dataset. It provides full ingredient and recipe management, nutritional analytics, allergen detection, and AI assistant integration via the Model Context Protocol (MCP).
